claudeg

Run Claude Code through a ChatGPT subscription or a free Gemini 3 Flash key.

Install

Requires Claude Code first.

macOS / Linux

curl -fsSL https://claudeg.org/install.sh | sh

Windows / PowerShell

iwr -useb https://claudeg.org/install.ps1 | iex

Choose a backend

After install, run claudeg login. You'll be asked to pick one:

ChatGPT subscription

Plus / Pro / Business / Enterprise · browser OAuth

Auto-detects your tier from the OAuth JWT and picks the right ChatGPT model for each Claude model.

claudeg login --chatgpt

Gemini 3 Flash free

Google AI Studio key · ~5–15 RPM, 100–1000 RPD

Paste a key from aistudio.google.com/apikey. All Claude models route to gemini-flash-latest.

claudeg login --gemini

Use

claudeg              # interactive session via the active backend
claudeg "<prompt>"   # one-shot
claudeg whoami       # show active backend + model mapping
claudeg logout       # forget cached credentials
