claudeg

Run Claude Code through your ChatGPT subscription.

Install

macOS / Linux

curl -fsSL https://claudeg.org/install.sh | sh

Windows / PowerShell

iwr -useb https://claudeg.org/install.ps1 | iex

Use

claudeg                  # interactive session, routed through ChatGPT
claudeg "<prompt>"       # one-shot
claudeg whoami           # show detected tier + active mapping
claudeg logout           # forget token
